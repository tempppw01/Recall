# v1.3 移动端回归清单（iOS / Android）

> 目标：验证 v1.3 阶段的移动端关键体验稳定，不出现“底部黑块 / 额外上拉 / 不可点击 / 同步失败难定位”。

## A. 设备与环境

- iOS Safari（建议 iPhone 刘海屏）
- Android Chrome（手势导航）
- 网络：Wi-Fi + 4G/5G 各一次（可选）

## B. 关键路径检查项

### 1) 布局与安全区

- [ ] 打开首页后，无底部黑块
- [ ] 主内容无需“先上拉”才对齐
- [ ] 左侧栏最后一项可见且可点击
- [ ] 右侧详情抽屉最后一项可见且可点击
- [ ] 设置弹窗底部按钮可点击

### 2) AI 助手（记录助手）

- [ ] 输入一个复杂任务后，返回 `reply + guidance + items`
- [ ] guidance 显示 2-4 条，内容可执行
- [ ] items 可单条加入待办
- [ ] 一键全部添加可用

### 3) 管理助手

- [ ] 发送“帮我挑出今天最该做的任务”可正常返回推荐
- [ ] 推荐项点击可定位到任务
- [ ] 一键建议（优先级/置顶/日期）可执行

### 4) 云同步

- [ ] Redis 参数正确时，push/pull/sync 可完成
- [ ] 断网时提示“网络异常”而非笼统失败
- [ ] Redis 缺失时提示“配置不完整”
- [ ] 服务端异常时提示“服务异常/稍后重试”
