# v0.3 任务清单（仅规划，禁止开工）

> 前提：只有在 v0.2 完成并完成对应发版后，才允许切换到 v0.3。

## 目标

- 提升同步可靠性与问题定位效率，减少多端状态不一致的不确定性。

## 任务列表

### A) 同步模型与冲突策略

- ⬜ 明确 sync 模型与冲突策略（DoD：文档清晰描述成功 / 冲突 / 覆盖 / 重试路径）
- ⬜ 返回冲突摘要与可执行提示（DoD：接口输出可直接指导用户下一步操作）

### B) 可观测性与诊断

- ⬜ 统一错误码 / requestId / 结构化日志（DoD：关键 API 可按入口、阶段、结果排查）
- ⬜ 增加深度健康检查（DoD：默认浅检查，深度模式覆盖 DB / sync 关键依赖）
