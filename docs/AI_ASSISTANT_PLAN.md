# AI Assistant Capsule Switch Plan

目标：在 AI 助手页面顶部增加胶囊切换按钮，可在两种助手间切换：

- **AI 记录助手**：专注记录对话与整理（现有 todo-agent 交互）。
- **AI 管理助手**：可调取当前所有任务列表，给出优先级/推荐/建议。

## Phase 1（可用版）

- [ ] 顶部胶囊按钮：记录助手 / 管理助手
- [ ] 管理助手：将当前任务列表摘要传给后端（仅必要字段）
- [ ] 新增后端 mode：`manage-agent`（返回 reply + recommendations）
- [ ] 管理助手 UI：展示 AI 回复 + 推荐列表（可点击打开任务详情）
- [ ] 构建通过（npm run build）
- [ ] commit + push

## Phase 2（增强版）

- [ ] 管理助手支持：只看未完成 / 只看今日 / 只看逾期 的快捷筛选
- [ ] 推荐支持“直接操作”：置顶/改优先级/改日期（需要明确动作协议）
