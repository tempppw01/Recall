# v0.1.2 日历视图 UI 修复清单

> 目标：把日历相关界面的明显可见问题收敛一轮，不做大规模架构改造。

## 一、搜索输入与候选框

- [x] 修复选择城市后，输入框回填触发重复搜索的问题
- [x] 修复候选结果中同一城市重复出现的问题
- [x] 修复已选城市状态下仍显示“未找到匹配城市”的误导性提示
- [x] 增加候选框显示条件控制：仅在“正在搜索”或“存在候选结果”时展示
- [x] 增加点击输入框/切换视图后的状态一致性检查

## 二、天气卡片与输入状态一致性

- [x] 已选城市后，天气卡片与输入框城市标签保持一致
- [x] 切换城市时，旧天气状态不要短暂误导展示
- [x] 网络失败时提示文案更明确，不与“无结果”混淆

## 三、不同日历视图状态残留

- [x] 月视图 / 周视图 / 日视图 / 日程视图切换时检查状态残留
- [x] 切换视图后避免候选框残留在错误位置
- [x] 切换日期后确认天气展示和标题同步更新

## 四、布局与细节

- [x] 修复小屏幕下输入框、候选框、天气卡片的排版问题
- [x] 修复文本溢出、边距不齐、视觉层级不清晰的问题
- [x] 检查空态文案是否准确，不制造误导

## 五、验证清单

- [x] 输入“北京”并选择后，不再弹出重复候选
- [x] 输入不存在城市时，提示正确
- [ ] 已选择城市时，不显示误导性空结果提示
- [ ] 切换月/周/日/日程视图后，输入和天气状态正常
- [ ] 移动端下布局不遮挡、不重叠
