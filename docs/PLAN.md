# 计划与发布文档入口

所有计划 / 审计 / 发布记录统一放在：`docs/plan/`

- 计划总览（当前版本、进度统计、导航）：`docs/plan/README.md`
- 全局路线图：`docs/plan/roadmap.md`
- 当前版本任务清单（唯一事实来源）：`docs/plan/versions/0.0.1.md`
- 后续版本规划：`docs/plan/versions/0.0.2.md` / `docs/plan/versions/0.0.3.md` / `docs/plan/versions/0.0.4.md`
- 发布历史（与 GitHub Release 对应）：`docs/plan/releases/CHANGELOG.md`

> 约束：仓库根目录与 `docs/` 根目录不再新增散落的 TODO / PLAN 文件；历史计划统一迁移到 `docs/plan/archive/`。
